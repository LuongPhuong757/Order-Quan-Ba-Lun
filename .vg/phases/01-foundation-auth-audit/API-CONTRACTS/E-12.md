## E-12 GET /admin/audit

**Decision:** P01.D-07 + D-28 (meta-audit logged)
**Auth:** OwnerGuard

### Query
```ts
?actor=<id>&action_kind=<string>&from=<ts_ms>&to=<ts_ms>&page=1&page_size=20
```

### Response 200
```ts
export const AuditListResponse = z.object({
```
