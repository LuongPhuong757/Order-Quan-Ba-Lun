## E-09 GET /admin/users

**Decision:** OwnerGuard
**Auth:** OwnerGuard

### Query
```ts
?page=1&page_size=20  // default; max page_size=100
```

### Response 200
```ts
export const UsersListResponse = z.object({
  data: z.object({
    items: z.array(z.object({
      id: z.string(),
      username: z.string(),
      is_active: z.boolean(),
      is_owner: z.boolean(),
      created_at: z.number()
    })),
    total: z.number(),
    page: z.number(),
    page_size: z.number()
  })
});
```

### Errors
- 403 `ADMIN_REQUIRED`
